Document Cross-References

Build a directed reference network across documents in this project. For each document, this task scans all tags for the selected additional information key (e.g. transkribus_doc_id), groups the mentions by target document, and writes a structured connection block into the document metadata.

  • Connections are directed (source → target) and weighted by mention count.
  • Target documents not found in this project are recorded with in_corpus: false.
  • Re-running the task replaces the existing connection block entirely.
  • Documents with no matching tags receive an empty connection block.
